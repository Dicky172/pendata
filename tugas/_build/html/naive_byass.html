Naive Baiyes#

Naive Bayes adalah metode klasifikasi data yang menggunakan prinsip probabilistik (peluang) untuk memprediksi kategori suatu data berdasarkan fitur-fitur yang dimilikinya.

Kenapa disebut ‚Äúnaive‚Äù (polos)? Karena metode ini berasumsi bahwa semua fitur data tidak saling berhubungan (independen). Contohnya, jika kita mengklasifikasi buah berdasarkan warna, bentuk, dan rasa, Naive Bayes akan menganggap warna tidak memengaruhi bentuk atau rasa, meskipun kenyataannya mungkin ada hubungan (misalnya, buah merah biasanya bulat). Asumsi ini membuat perhitungannya jauh lebih sederhana dan cepat.

Contoh Penerapan: Misalnya, untuk membedakan email ‚Äúspam‚Äù dan ‚Äúbukan spam‚Äù:

Fitur yang digunakan bisa berupa kata-kata dalam email (misalnya ‚Äúhadiah‚Äù, ‚Äúgratis‚Äù).

Naive Bayes menghitung peluang email mengandung kata-kata tersebut dan menentukan apakah itu spam, tanpa memperhatikan hubungan antar kata (misalnya, ‚Äúhadiah‚Äù dan ‚Äúgratis‚Äù dianggap tidak terkait).

Kelebihan:

Cepat dan efisien, cocok untuk data besar atau data sederhana.

Bisa bekerja dengan baik meskipun asumsi ‚Äúindependen‚Äù tidak sepenuhnya benar.

2. Rumus Na√Øve Bayes#

Na√Øve Bayes menggunakan rumus:

\[ P(C|X) = \frac{P(X|C) P(C)}{P(X)} \]

Dimana:

( P(C|X) ) adalah probabilitas kelas ( C ) diberikan fitur ( X ) ( P(X|C) ) adalah probabilitas fitur ( X ) muncul dalam kelas ( C ) ( P(C) ) adalah probabilitas prior dari kelas ( C ) ( P(X) ) adalah probabilitas total fitur ( X )

Karena kita membandingkan antar kelas, kita hanya fokus pada pembilang dari rumus tersebut:

\[ P(X|C) P(C) \]

3. Menghitung Mean dan Variance untuk Setiap Kelas#

3. Menghitung Probabilitas dengan Distribusi Gaussian#

Formula distribusi Gaussian: $\( P(x|class) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \)$

Untuk Petal Length = 4.7

\[ \begin{align}\begin{aligned} P(4.7 | Setosa) $$ = $$ \frac{1}{\sqrt{2\pi (0.1737^2)}} e^{-\frac{(4.7 1.462)^2}{2(0.1737^2)}} $$ ‚âà 0 $$ P(4.7 | Versicolor) $$ = $$ \frac{1}{\sqrt{2\pi (0.4699^2)}} e^{-\frac{(4.7 - 4.26)^2}{2(0.4699^2)}} $$ ‚âà 0.6545 $$ P(4.7 | Virginica) $$ = $$ \frac{1}{\sqrt{2\pi (0.5519^2)}} e^{-\frac{(4.7 - 5.55)^2}{2(0.5519^2)}} $$ ‚âà 0.3913\\**Untuk Petal Width = 1.8**\\ $$ P(1.8 | Setosa) $$ = $$ \frac{1}{\sqrt{2\pi (0.1054^2)}} e^{-\frac{(1.8 - 0.246)^2}{2(0.1054^2)}} $$ ‚âà 0 $$ P(1.8 | Versicolor) $$ = $$ \frac{1}{\sqrt{2\pi (0.1978^2)}} e^{-\frac{(1.8 - 1.325)^2}{2(0.1978^2)}} $$ ‚âà 0.1937 $$ P(1.8 | Virginica) $$ = $$ \frac{1}{\sqrt{2\pi (0.2747^2)}} e^{-\frac{(1.8 - 2.026)^2}{2(0.2747^2)}} $$ ‚âà 1.306\\---\\#### 4. Menghitung Probabilitas Akhir untuk Setiap Kelas Formula: $$ P(Class | Data) \propto P(Class) \times P(x_1 | Class) \times P(x_2 | Class) \end{aligned}\end{align} \]

# Data input untuk klasifikasi
input_petal_length = 4.7
input_petal_width = 1.8

# Menghitung probabilitas prior (P(Class))
class_counts = df["class"].value_counts()
total_samples = len(df)
prior_probs = class_counts / total_samples

# Menghitung mean dan standar deviasi untuk setiap kelas
stats = df.groupby("class").agg({
    "petal_length": ["mean", "std"],
    "petal_width": ["mean", "std"]
})

# Rename kolom hasil aggregasi agar lebih mudah diakses
stats.columns = ["petal_length_mean", "petal_length_std", "petal_width_mean", "petal_width_std"]

# Fungsi untuk menghitung probabilitas berdasarkan distribusi Gaussian
def gaussian_prob(x, mean, std):
    return norm.pdf(x, mean, std) if std > 0 else 0  # Hindari std = 0

# Menghitung likelihood untuk setiap kelas
likelihoods = {}
for species in stats.index:
    p_length = gaussian_prob(input_petal_length, stats.loc[species, "petal_length_mean"], stats.loc[species, "petal_length_std"])
    p_width = gaussian_prob(input_petal_width, stats.loc[species, "petal_width_mean"], stats.loc[species, "petal_width_std"])
    likelihoods[species] = prior_probs[species] * p_length * p_width  # P(Class) * P(X|Class)

# Menentukan kelas dengan probabilitas tertinggi
predicted_species = max(likelihoods, key=likelihoods.get)

# Output hasil klasifikasi
print(f"Data ({input_petal_length}, {input_petal_width}) diklasifikasikan sebagai: {predicted_species}")
